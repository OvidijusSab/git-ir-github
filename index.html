Git

Git'as yra populiariausia kodo versijavimo programa.

Git komandos

  • init - sukuria git repozitorija dabartiniame aplankale
  • status - nurodo informacija apie lokalios gito repozitorijos statusa.g
  • add - prideda nurodytus failus prie commitui paruostu failu (stage'ina)
  • rm --cached isima nurodytus failus is stageinimo zonos
  • commit -m "zinute" - sukuria kodo versija is stageintu failu ir prideda zinute apie pakeitimus (zinute turetu buti trumpa(ne ilgesne nei 50 simboliu ) ir aiski)
  • restore - grazina nurodytus failus i buvusio commito busena ( panaikina naujus pakitimus padarytus po paskutinio commito
